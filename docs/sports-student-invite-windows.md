# 學生賽事 — 官方邀請窗口名單（運動頻道拉式投稿用，隨時更新）

運動頻道＝**學生賽事、純拉式（投稿＋邀請）**。本清單供編輯用 `sports-invite-draft` skill 對**官方機構窗口**起草邀請信、**人工送出**，邀請其提供賽事資訊投稿（`/sports/submit/`）。

> 鐵則：**只列機構/聯賽/協會層級的公開官方窗口**，不收個別老師/教練/學生/未成年個資；不自動冷寄，一律人工確認後送出。最後盤點：2026-06-21（實測官網可達性與公開窗口）。
> ⚠️ 名稱更正：教育部「體育署」2025/9 已升格為**運動部**（下設全民運動署）；賽事規程現以運動部核定函發出。舊稱勿用。`sports.url.tw` 為測試頁、非官方入口，勿引用。

## 實務建議
- 邀請優先：**高中體總、大專體總、學生棒球聯盟**（一窗涵蓋多賽，最有效率）；其餘協會作單項補強。
- **不列承辦學校與屆次官網**（全中運/全大運每年換承辦、網域一次性、無長期窗口）；要長期合作對口上層總會即可。
